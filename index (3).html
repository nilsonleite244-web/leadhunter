Configuração da API

Variáveis de ambiente necessárias no Railway:

# Obrigatórias
DATABASE_URL=postgresql://...
EVOLUTION_URL=https://evolution.railway.app
EVOLUTION_KEY=sua-api-key
EVOLUTION_INSTANCE=leadhunter
# Opcionais
RESEND_API_KEY=re_xxx
EMAIL_FROM[email protected]
Configurações Anti-Ban WhatsApp

Configure essas variáveis no Railway (worker-whatsapp.js). Valores conservadores = mais seguro.

100 = seguro · 200 = moderado · 300+ = arriscado

⚡ Estratégia de escala:
Cada cliente conecta seu próprio número → cada número = 100 msgs/dia. Com 10 clientes = 1.000 msgs/dia totais, sem custo extra.
